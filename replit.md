# GiftBot — Telegram Mini App

## Описание
Telegram-бот с мини-приложением (Mini App) для игр, розыгрышей, кейсов и магазина подарков. Интерфейс на русском языке.

## Структура проекта
- `bot.js` — основной файл: Express-сервер + Telegram-бот (telegraf)
- `public/` — фронтенд мини-приложения (HTML/CSS/JS, статика)
- `db.json` — локальная база данных (создаётся автоматически)

## Технологии
- **Node.js 18+**
- **Express** — HTTP-сервер, отдаёт статику и API
- **Telegraf** — библиотека для Telegram Bot API
- **Jimp** — генерация изображений
- **Telegram Web App JS** — интеграция с Telegram Mini App

## Конфигурация
- Сервер слушает на `0.0.0.0:5000`
- `BOT_TOKEN` — токен Telegram-бота (обязательный секрет)
- `APP_URL` — URL для webhook (опционально; без него бот работает в режиме polling)

## Запуск
```bash
npm install
node bot.js
```

## Деплой
Тип деплоя: **VM** (бот должен работать постоянно для polling).
