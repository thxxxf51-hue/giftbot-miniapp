# GiftBot — Telegram Mini App

## Описание
Telegram-бот с мини-приложением (Mini App) для игр, розыгрышей, кейсов и магазина подарков. Интерфейс на русском языке.

## Структура проекта
- `bot.js` — основной файл: Express-сервер + Telegram-бот (telegraf)
- `public/` — фронтенд мини-приложения (HTML/CSS/JS, статика)
  - `public/js/admin.js` — логика административной панели
  - `public/css/style.css` — стили приложения
  - `public/index.html` — основной HTML
- `db.json` — локальная база данных (создаётся автоматически)

## Технологии
- **Node.js 18+**
- **Express** — HTTP-сервер, отдаёт статику и API
- **Telegraf** — библиотека для Telegram Bot API
- **Jimp** — генерация изображений
- **Telegram Web App JS** — интеграция с Telegram Mini App

## Конфигурация (секреты)
- `BOT_TOKEN` — токен Telegram-бота (обязательный)
- `GITHUB_PERSONAL_ACCESS_TOKEN` — токен GitHub для резервного копирования и пушей
- `APP_URL` — URL для webhook (для Railway/деплой); без него бот работает в polling

## Railway (production)
- Деплой автоматический с GitHub (ветка main)
- База данных восстанавливается из ветки `db-backup` при каждом старте
- Перед запуском сервера происходит restore из GitHub → потеря данных при редеплое исключена
- Нужно задать `GITHUB_PERSONAL_ACCESS_TOKEN` и `APP_URL` в Railway Environment Variables

## Команды бота (только для админа)
- `/repair` — включить/выключить режим тех. работ (только в личке)
- `/cpromo КОД СУММА КОЛ-ВО` — создать промокод
- `/vpromo КОД СУММА КОЛ-ВО` — создать VIP промокод
- `/promos` — список промокодов
- `/ban @username` — забанить
- `/unban @username` — разбанить
- `/banlist` — список банов

## Административная панель
- Доступна только для UID `6151671553`
- Дашборд, Юзеры (с профилем по клику), Задания, Розыгрыши, Промокоды, Уведомления
- Профиль пользователя: баланс, звёзды, рефералы, VIP, корона, инвентарь, последние транзакции

## Запуск (разработка)
```bash
npm install
node bot.js
```
